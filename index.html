Awesome Sauce

sauce is a system for processing and altering program output. The simplest application is to colorize shell output, but it can also be used to process logs and alert on unfamiliar entries, correlate events across different sources or create a static blog.

Features:

  • Pluggable and flexible matching
    • Patterns are defined as regexes, actions are pure code
  • Matching / classification and processing are separate activities
  • Easily swap output formats for filetype translation

Future:

  • Weighting for matchers
  • Defaults for all syslogs, shell history / output, SCMs, etc.
